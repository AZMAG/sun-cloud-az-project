Test the API

Overview

NIRA provides an OpenAPI 3.0 specification file that can be used with your favorite testing program. The tool we will use is Postman.
Additionally, you will need a tool for decoding the protobuf response files. We will use Protoc.

Test with Postman

  • Download API definition here .
  • Download Postman.
  • Import the API definition into Postman.
    • During import click on “Show advanced settings” and change the following:
      • “Naming requests” to URL.
      • “Request parameter generation” to Schema.
      • “Folder organization” to Tags.
  • Use the Authentication endpoint /user/login to get a valid AUTH_SESSION cookie.
  • Get your Data Bundle Permissions you with the /user/data-bundles endpoint.
  • Fetch data with one of the APIs described here.
    • Save the response to a file called response.bin.
  • Decode response.bin with the necessary protobuf file described here.
