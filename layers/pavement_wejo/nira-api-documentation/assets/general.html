General

Welcome to the documentation of NIRA’s Road Surface Information API.

You can directly view the endpoint documentation here.
To test the API, read here.

Technical Terms

  • Base url: The URL prefix for each api endpoint.
  • Link: A stretch of road in the road geometry.
    • Link id: A unique id assigned to each link.
  • Sub-link id: Identifier for a specific part of a link.
  • Offset: Identifies a specific part of a link (similar to sub-link) but is more dynamic.
  • Tile: A defined area boundary in the world, read more.
  • Data bundle: A named bundle of data values that NIRA provides, correlated to the road geometry.
